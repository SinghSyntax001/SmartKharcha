# **App Name**: SmartKharcha AI

## Core Features:

- Profile Creation: Create and store user profiles with relevant financial information.
- AI Chatbot: Implement a chatbot powered by Groq Cloud to provide personalized financial advice based on user profiles, retrieved documents and deterministic facts; this tool uses a chain of reasoning and acts based on all the information at hand.
- Knowledge Base Retrieval: Retrieve relevant financial documents from a seed KB based on user questions and profile data.
- Deterministic Calculations: Calculate key financial metrics (e.g., insurance coverage, investment returns) based on user profile data.
- Recommendation Display: Display AI-generated advice with confidence scores and sources from the knowledge base.
- Fallback System: Provide deterministic answers and cite KB sources when Groq API key is missing or API call fails.
- User Data Persistence: Save user profiles locally in the browser to persist sessions across reloads.

## Style Guidelines:

- Primary color: Deep blue (#1A5276) to inspire confidence and trust.
- Background color: Light gray (#F5F5F5) to provide a clean and modern look.
- Accent color: Bright teal (#2ECC71) to highlight key information and calls to action.
- Body font: 'PT Sans', a humanist sans-serif suitable for headlines or body text.
- Headline font: 'Playfair', a modern serif similar to Didot with a high-end feel. Note: currently only Google Fonts are supported.
- Use clean and modern icons to represent various financial concepts.
- Subtle animations and transitions to provide a smooth user experience.
